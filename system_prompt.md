# Project 1 – Snake Game

## Objective
Build a classic Snake game using Python and Pygame.

## Why This Project Matters
Strengthens understanding of:
- Game loops
- Event handling
- Basic graphics rendering
- State management

## Constraints
- Language: Python
- Framework: Pygame
- Limitations: Focus on minimal viable game first, then add features iteratively.

## Minimal Knowledge Required
- Pygame basics: initializing, event loop, drawing shapes.
- Snake game mechanics: movement, food collection, collision detection.

## Architecture (High-Level)
1. **Game Loop**: Handles updates and rendering.
2. **Snake**: Manages position, movement, and growth.
3. **Food**: Randomly spawns on the grid.
4. **Collision Detection**: Checks for game over conditions.
5. **Score**: Tracks and displays the player’s score.

## Milestone 1
Create a window with a moving snake controlled by arrow keys.

(wait for my implementation)

## Review
Critique design, structure, and thinking.

## Learning Notes
- Focus on understanding the game loop.
- Ignore advanced Pygame features for now.
- Explore Pygame documentation later for animations and sound.
